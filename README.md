This is an excercise to display random greetings messages using React-Rails gem for Microverse.

## Getting Started

### Installation

1. Clone the repo

   git clone https://github.com/smunozmo/hello-rails-react.git
  

### Usage

1. Install ruby gems dependencies

   ```sh
   bundle install
   ```

2. Install node dependencies

   ```sh
   npm install
   ```

3. Set up database

   ```sh
   rails db:create db:migrate db:seed
   ```

4. Run the web server

   ```sh
   rails server
   ```

5. Access to the website in a browser with the following link

   ```sh
   localhost:3000
   ```
### Test

For testing the classes run:

   ```sh
   bundle exec rspec spec
   ```
